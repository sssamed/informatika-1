Osnove CSS-a

CSS (Cascading Style Sheets) je stilski jezik pomocu kojeg upravljamo izgledom HTML stranice. Dozvoljava dodavanje stilova, poput boja, fontova i margina HTML dokumentu i njegovim elementima.

CSS Selektori

Selektor Opis
element Primijeni stil na sve elemente odredjenog tipa u dokumentu (body, h1, p...)
.class Primijeni stil na sve elemente jedne klase (.navigation, .hero-section...)
#id Primjenjuje stil na element koji ima odredjeni ID (#naslov, #html-sekcija...)

CSS Svojstva

  • color: Podesava boju teksta
  • font-size: Podesava velicinu fonta
  • margin: Podesava marginu oko elementa
  • padding: Podesava padding unutar elementa (udaljenost izmedju elementa i bordera)

U CSS-u postoje i pseudoklase koje koristimo za dodavanje stila elementu nakon necega sto se desi s tog elementa.
Slika ispod ima pseudoklasu hover koja mijenja stil slike dok mis lebdi iznad nje. Dodata je i tranzicija kako bi promjena izgledala ljepse.
