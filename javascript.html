Uvod u JavaScript

JavaScript je interpretirani programski jezik visokog nivoa koji se koristi za dodavanje funkcionalnosti web stranicama.
Postoje mnogi framework-ovi koji prosiruju primjenu JavaScript-a poput Angular, NodeJS, React...

Korisne naredbe u JavaScriptu

  • Varijable: Koristimo kljucne rijeci var, let, i const za deklaraciju varijabli.
  • Funkcije: Mozemo praviti funkcije koje sadrze dio koda koji bi mogli ponovo upotrijebiti.
    Koristimo kljucnu rijec function, zatim dajemo ime funkciji i stavljamo parametre u zagrade.
  • Uslovi: Koristimo if i else kada radimo sa uslovima.
  • Petlje: Sluze nam za iteraciju, odnosno ponavljanje dijela koda, kao sto su for i while.
  • Nizovi: Spremaju niz vrijednosti na jedno mjesto. Elemente niza pisemo unutar [], a on se deklarise kao i obicna varijabla
  • Objekti: U JavaScript-u imamo i OOP (objektno orijentisano programiranje), a za objekte koristimo JSON (JavaScript Object Notation)
    JSON se koristi i u drugim programskim jezicima za zapisivanje podataka.
